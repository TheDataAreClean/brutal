## og-image.png / favicon.png

Auto-generated by `build.py` (requires Pillow). Do not manually place images here — they will be overwritten on the next build.

The seasonal accent colour (one of 12 Bengaluru bloom colours, based on the current month) is applied to both images at build time.

## fonts/MaterialSymbolsSharp.woff2

Downloaded automatically by `build.py` on each build via the Google Fonts `icon_names` subsetting API. Contains only the 22 icons used across the site (~25KB vs ~2MB for the full variable font). To add a new icon, add its name to `ICON_NAMES` in `build.py` and rebuild.

## fonts/SchibstedGrotesk.ttf

Used by Pillow at build time to render text in the OG image. The web font is loaded separately from Google Fonts CDN.
