## og-image.png / favicon.png

Both are generated by `build.py` on every build (requires Pillow). Do not manually place images here — they will be overwritten.

- **og-image.png** — seasonal; copied from `assets/monthly/og-<month>.png`. Pre-generate all 12 with `python3 build.py --gen-monthly`.
- **favicon.png** — static design (white background, black border box, black square at bottom-left). Generated fresh on every build using the current month's accent colour for `generate_favicon()`. Not seasonal — the design is fixed.

## fonts/MaterialSymbolsSharp.woff2

Downloaded automatically by `build.py` on each build via the Google Fonts `icon_names` subsetting API. Contains only the icons used across the site (~25KB vs ~2MB for the full variable font). To add a new icon, add its name to `ICON_NAMES` in `build.py` and rebuild.

## fonts/SchibstedGrotesk.ttf

Used by Pillow at build time to render text in the OG image. The web font is loaded separately from Google Fonts CDN.

## playground/

Screenshot previews for playground project cards on the play page. One PNG per project, referenced by filename in `content/play/playground.md`. Not auto-generated — drop files here manually.
